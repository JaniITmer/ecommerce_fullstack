# 🛒 EcommerceApp

A full-stack e-commerce web application built with ASP.NET Core 10 and Next.js.
Developed as a portfolio project to demonstrate real-world backend and frontend skills.

## Tech Stack

**Backend:** C# · ASP.NET Core 10 · Entity Framework Core 
**Frontend:** Next.js · TypeScript · Tailwind CSS 
**Auth:** JWT · Role-based access control  
**API Docs:** Scalar (OpenAPI 3.1)  
**DevOps:** Docker · Docker Compose  

## Features

- [ ] Product listing with filtering, search and pagination
- [ ] Product detail page with reviews
- [ ] Shopping cart
- [ ] User registration and login (JWT auth)
- [ ] Checkout flow with order summary
- [ ] Order history
- [ ] Admin panel: manage products and orders

## Getting Started

### Prerequisites
- .NET 10 SDK
- Node.js 20+
- Docker Desktop

### Run backend locally

git clone https://github.com/JaniITmer/ecommerce_fullstack.git
cd ecommerce_fullstack/backend/EcommerceApi
dotnet run

API: https://localhost:7092  
API Docs: https://localhost:7092/scalar/v1
